# Minesweeper
Mine sweeper geschreven in C.  

## Spelen
Je bestuurt de cursor met de pijltjes toetsen.  
Je kan een `cell` *flaggen* met `f` en *openen* met `enter`.  
Wanneer je het spel wilt verlaten, druk je simpelweg op `esc` of op `q`.
